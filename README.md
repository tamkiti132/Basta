## ▫️ 使用技術

バックエンド

-   PHP 8.1.13
-   Laravel 9.52.5
-   Livewire 2.12.3
-   Jetstream 3.0.2
-   Laravel Socialite（Google OAuth 2.0 API を使用）

フロントエンド

-   HTML
-   CSS
-   Tailwind CSS 3.3.1
-   Alpine.js 3.12.0
-   JavaScript
-   Vite 4.2.1

データベース

-   開発環境: MySQL 5.7.39（MAMP）
-   本番環境: MySQL 8.0

インフラ

-   開発環境: Apache 2.4.54（MAMP）
-   本番環境: Apache 2.4.62

デザイン

-   Draw\.io

その他

-   PHPUnit, Git, GitHub
